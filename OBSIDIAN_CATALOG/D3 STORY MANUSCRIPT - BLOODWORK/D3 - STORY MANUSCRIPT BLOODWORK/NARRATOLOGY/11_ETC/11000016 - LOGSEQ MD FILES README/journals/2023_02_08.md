- # Learn Game Logic
	- There are several books that are widely considered to be some of the best resources for learning game logic and game development:
	- "Game Programming Patterns" by Robert Nystrom
	- "Game Engine Architecture" by Jason Gregory
	- "Programming Game AI by Example" by Mat Buckland
	- "Introduction to Game Design, Prototyping, and Development" by Jeremy Gibson and Kate Hopper
	- "The Art of Game Design: A Book of Lenses" by Jesse Schell
